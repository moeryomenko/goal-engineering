# Contributing

Thanks for helping make this the go-to reference for Grok Build `/goal`.

## High-value contributions

- New goal patterns with example `/goal` lines
- goal-audit / goal-init / goal-cost improvements
- Verifier skill improvements
- Real stories in `stories/` (wins and honest failures)
- Cross-tool notes in `docs/primitives-matrix.md`

## Process

1. Fork and branch
2. Run `npm test` and `npm run validate` from repo root
3. Open PR with clear description
4. Link related issues or blog posts if any

Discuss ideas in [GitHub Discussions](https://github.com/cobusgreyling/goal-engineering/discussions).
